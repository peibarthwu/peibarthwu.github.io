Guiville

Worldbuilding, Net.Art (2022-Ongoing)

Guiville (pronounced 'gooey-ville') is a city within a graphical user interface or GUI, a visual language developed in the late 1970s-early 1980s that made navigating through a file system easier. All the roads and houses in Guiville may be moved by anyone, which is a HUGE inconvenience, because sometimes a citizen might find that all the roads are suddenly rearranged (or worse, their entire house is moved). Guiville imagines how user interfaces (which are generally supposed to increase convenience and make navigation easier) could be incredibly incovenient and make navigation difficult. (What if a citizen's entire house gets moved by someone playing around with Guiville from a computer?) More broadly, this project explores urban planning and development, space and runtime constraints, acoustic ecology, and life on the internet.
